# 8D AI Solution Engineering Framework

**8D** is the primary user journey of the [AI Solution Engineering Playbook](https://sudip-ai-playbook.github.io/). Practitioners navigate **Define → Deliver**. The legacy 20-stage ConsultAI lifecycle remains available as detailed tools under each dimension.

## The eight dimensions

| Stage | Purpose | Stage gate |
| --- | --- | --- |
| **Define** | Problem, decision, outcome, metrics, stakeholders, scope—without naming an AI product | Problem expressed without a particular AI product or model |
| **Discover** | Business / systems / team / AI lenses; maturity and opportunity capture | Evidence-backed opportunity list exists |
| **Diagnose** | Root cause, MECE, pain-to-value, prioritisation themes | Every proposed solution addresses a verified cause |
| **Design** | To-be workflow, human–AI allocation, architecture, operating model | Workflow and allocation documented before tech lock-in |
| **De-risk** | Risk, ethics, privacy, controls with owners—not end-of-project paperwork | Owner, control, evidence, residual risk for each material risk |
| **Demonstrate** | Pilot / proof with evaluation evidence—not demo theatre | Go / modify / stop recommendation with evidence |
| **Decide** | Business case, executive narrative, explicit ask, decision log | Explicit ask and decision log entry |
| **Deliver** | Roadmap horizons, adoption, ops, benefits, closure | Approved roadmap and benefits plan with owners |

## Required outputs (summary)

| 8D | Typical artefacts |
| --- | --- |
| Define | Problem statement; decision; outcome; metrics; stakeholders; scope |
| Discover | Discovery lenses; opportunity list; maturity notes |
| Diagnose | Root-cause map; quantified pain; hypotheses; themes |
| Design | To-be; options; recommendation; oversight; architecture; dependencies |
| De-risk | Risk register with controls and residual risk |
| Demonstrate | Pilot charter; evaluation evidence; go/modify/stop |
| Decide | Brief; business case; decision log; explicit ask |
| Deliver | Roadmap; cadence; benefits; change; handover |
