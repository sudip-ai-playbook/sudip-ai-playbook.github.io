# Final competency test

You are operating as a strong AI Solution Engineer when you can answer all of these for a live opportunity:

1. What business problem are we solving?
2. Why is AI appropriate?
3. Who receives the value?
4. How will value be measured?
5. What data is required?
6. Is that data usable and legally accessible?
7. Which AI pattern should be used?
8. Which model and platform should be selected?
9. How will the solution integrate with existing systems?
10. How will identity and access be controlled?
11. How will prompt injection and data leakage be prevented?
12. How will output quality be evaluated?
13. Where is human oversight required?
14. How will the system scale?
15. What will it cost to build and operate?
16. What are the alternatives?
17. What are the major risks and dependencies?
18. How will the solution be governed?
19. How will users adopt it?
20. How will it be supported in production?
21. How will benefits be tracked?
22. What decision does the client need to make now?

<Checklist title="Self-assessment">

- [ ] I can answer each question in writing for my current case
- [ ] Facts, assumptions and hypotheses are labelled
- [ ] Every “I don’t know” has an owner and a date to close it
- [ ] The client decision (Q22) is explicit

</Checklist>
